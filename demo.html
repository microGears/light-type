Buttons

To create LightType button, add class lt-button to any element, as below.

link(lt-button) span(lt-button)
div(lt-button)

Code example:

                            link(lt-button)
                            span(lt-button)
                            
div(lt-button)

Buttons can be of different sizes, there are three basic: small, regular and large.
To change the size of a button, add to class lt-button class small or large, as shown below.

link(lt-button) span(lt-button)
div(lt-button)

Code example:

                            link(lt-button)
                            span(lt-button)
                            
div(lt-button)

Buttons can be of different colors, there are four basic: alert, success, warning and info.
To change the color of a button, add to class lt-button class lt-button-*, as shown below.

Alert
Success
Warning
Info

Code example:

                            
Alert

                            
Success

                            
Warning

                            
Info

To create LightType buttons bar, wrap any elements of class of lt-button into element class lt-button-bar, as below.

Link(lt-button) 1 Link(lt-button) 2 Link(lt-button) 3

Code example:

                    

                        Link(lt-button) 1
                        Link(lt-button) 2
                        Link(lt-button) 3
